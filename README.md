## FBS Reader

封装 Flatbuffers 操作函数，简化数据使用过程，提高数据安全性

**支持语言：**
- C++
- Python3


### 编译

**Linux or macOS:**
``` shell
    git clone https://github.com/google/flatbuffers.git && cd flatbuffers
    mkdir release && cd release
    cmake ..
    make && sudo make install
    cd ../.. && rm -rf flatbuffers
    
    mkdir build && cd build && cmake ..
    make
    make install
```
